# ARMgeddon Mobile Miner - Build Files

## Download Instructions

### Android APK
- **File**: `armageddon-mobile-miner.apk`
- **Size**: ~25MB
- **Requirements**: Android 6.0+
- **Installation**: Enable "Unknown Sources" in Android settings, then install APK

### iOS IPA
- **File**: `armageddon-mobile-miner.ipa`
- **Size**: ~30MB
- **Requirements**: iOS 12.0+
- **Installation**: Sideload using AltStore, Xcode, or enterprise distribution

## Build Commands

To generate these files:

```bash
# Install dependencies
npm install

# Build Android APK
npm run build:apk

# Build iOS IPA
npm run build:ios
```
