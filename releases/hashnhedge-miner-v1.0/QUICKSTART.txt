HashNHedge Miner v1.0 - Quick Start Guide
=========================================

EASY 3-STEP SETUP:

1. INSTALL (if needed):
   - Double-click "INSTALL.bat" if you don't have Python
   - Or just double-click "run_miner_dev.bat" to auto-install

2. START MINING:
   - Double-click "run_miner_dev.bat"
   - Enter your Solana wallet address
   - Click "Connect Wallet"
   - Click "Start Mining"

3. EARN HNH TOKENS:
   - Watch real-time stats
   - Earn 1 HNH per share found
   - Automatic payouts to your wallet

FILES INCLUDED:
===============
- run_miner_dev.bat      → Quick start (just double-click!)
- hashnhedge_miner_gui.py → Basic miner interface
- hashnhedge_miner_advanced.py → Advanced miner with tabs
- build_miner.bat        → Create standalone .exe files
- INSTALL.bat           → Install Python dependencies
- MINER_README.md       → Complete user manual
- logo.ico              → Application icon

SYSTEM REQUIREMENTS:
===================
- Windows 7/8/10/11
- Internet connection
- 100MB RAM minimum
- Any modern CPU

MINING DETAILS:
==============
- Algorithm: SHA256 proof-of-work
- Pool fee: 3% (automatic)
- Reward: 1 HNH token per valid share
- Difficulty: Auto-adjusting
- Performance: ~1-10 KH/s typical

SECURITY:
========
- Only your PUBLIC wallet address is needed
- NO private keys required
- Direct payouts to your wallet
- HTTPS secure connections

NEED HELP?
